📦 Deployment Guide — TrustChain.sol
1
Open Remix IDEGo to remix.ethereum.org → New File → name it TrustChain.sol → paste the contract code from the Contract tab above
2
CompileSolidity Compiler → version 0.8.20 → click Compile TrustChain.sol → no errors should appear
3
Start GanacheRun Ganache on localhost:7545 — Chain ID must be 1337. Import a Ganache account into MetaMask (use the private key from Ganache UI).
4
DeployDeploy & Run tab → Environment: Injected Provider - MetaMask → click Deploy → copy the deployed contract address shown in the Remix terminal
5
Connect FrontendClick Connect Wallet above → go to Admin tab → paste the contract address → start registering landlords and issuing agreements!

Enter Agreement ID

02

03
Result Returned From Blockchain
Tenant name, CNIC, property, rent, lease dates, issuer wallet, and Valid/Revoked status — all on-chain

🔧 Public Functions
Function Type Description
registerLandlord(address,string,string,string) write (owner) Register a new landlord wallet address on-chain
removeLandlord(address) write (owner) Remove a registered landlord
issueAgreement(string,string,string,string,uint256,uint256,uint256) write (auth) Issue a new tenancy agreement — callable by owner or landlord
revokeAgreement(string) write (auth) Mark agreement as revoked (record kept on-chain)
deleteAgreement(string) write (owner) Permanently delete agreement from contract storage
verifyAgreement(string) view (public) Fetch all fields of any agreement — no auth required
getAllAgreementIds() view (public) Returns array of all agreement IDs
getAllLandlords() view (public) Returns array of all landlord addresses
getLandlordInfo(address) view (public) Returns name, company, location for a landlord address
isRegisteredLandlord(address) view (public) Returns true if address is a registered landlord
transferOwnership(address) write (owner) Transfer admin rights to a new wallet address
